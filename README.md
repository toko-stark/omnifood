# Omnifood

Modern single-page marketing site for an AI-powered meal subscription.

> Built as part of the Jonas Schmedtmann **HTML5 & CSS3** course (required acknowledgment).

## Preview

- Live page markup: [index.html](index.html)
- Core styles: [css/general.css](css/general.css), [css/style.css](css/style.css), [css/queries.css](css/queries.css)
- Behavior: [js/script.js](js/script.js)

## Features

- Responsive hero, pricing, testimonials, and CTA sections
- Sticky navigation with smooth scrolling
- Mobile nav toggle with overlay
- Flex-gap Safari fallback detection
- Media-query-driven layout refinements

## Tech Stack

- HTML5, CSS3 (Grid/Flexbox, custom properties), Vanilla JS
- Google Fonts (Rubik), Ionicons

## Run Locally

1. Clone the repo:
   ```sh
   git clone <your-fork-url>
   cd omnifood
   ```
2. Open [index.html](index.html) in your browser (no build step needed).

## Project Structure

- `index.html` — landing page markup
- `css/` — base styles, components, responsive queries
- `js/script.js` — navigation behavior, smooth scroll, sticky header
- `img/` — logos, meals, gallery assets
